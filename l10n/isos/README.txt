This files are from Debian package iso-codes and copyright is for its respective
authors. To fetch the last version of this files execute in a Debian (based) OS:

    cd /tmp/
    mkdir isos_output
    apt-get source iso-codes
    cd iso-codes-****/iso-codes-****/
    ./configure --prefix=/tmp/isos_output/
    make
    make install
    cd /tmp/isos_output/share/locale/

And they they are.
